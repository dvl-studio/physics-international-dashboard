
# 物理系國際組｜招生與學習成效儀表板

本專案為物理系國際組招生與學習成效之靜態互動儀表板，使用 HTML、CSS 與 JavaScript 製作，並透過 GitHub Pages 發布。

## 專案內容

本儀表板包含：

- 國際組入學管道分析
- 來源高中與區域分布
- 入學成績中位數比較
- 第一學期修課負擔與成績表現
- 英文授課科目與學分占比
- 英文能力與英文授課參與之關聯分析

## 方法摘要

- 學生成績加權平均：Σ（課程成績 × 課程學分） ÷ Σ（納入計算的課程學分）
- 同課程標準化成績（Z-score）：以相同學年、學期及課程為單位，依所有修課學生的成績平均數與標準差進行標準化
- 組別比較：中位數、四分位數、Mann–Whitney U 與 Spearman 相關

## 資料說明

本儀表板僅呈現彙整分析結果，不嵌入學生姓名、學號或任何個人資訊。
